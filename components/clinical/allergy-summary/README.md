# Allergy Summary

### Import

```js
import AllergySummary from '@ltht-react/allergy-summary'
```

### Usage

```jsx
<AllergySummary allergy={allergy} />
```

### Properties

| Prop      | Required | Default | Type               | Description        |
| :-------- | :------- | :------ | :----------------- | :----------------- |
| `allergy` | Yes      |         | AllergyIntolerance | Allergy to display |
