# Timeline

### Import

```js
import Timeline from '@ltht-react/timeline'
```

### Usage

```jsx
<Timeline auditTrail={auditTrail} />
```

### Properties

| Prop         | Required | Default | Type                  | Description        |
| :----------- | :------- | :------ | :-------------------- | :----------------- |
| `auditTrail` | Yes      |         | `Maybe<AuditEvent>[]` | Timeline of events |
