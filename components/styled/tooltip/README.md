# Tooltip

<!-- STORY -->

### Import

```js
import Tooltip from '@ltht-react/tooltip'
```

### Usage

```jsx
<Tooltip content={'This is a tooltip'}>
  <div>Component That Needs Tooltip</div>
</Tooltip>
```

Control Placement

```jsx
<Tooltip content={'This is a tooltip'} placement="bottom">
  <div>Component That Needs Tooltip</div>
</Tooltip>
```
